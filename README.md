# StyleFolio

StyleFolio is an AI-powered web application that helps users manage their virtual wardrobe and generate outfit recommendations based on weather, events, and user preferences.

## 🚀 Features

- Add and categorize clothing items
- Generate outfit recommendations
- API integration for dynamic data (weather, etc.)
- Responsive design for all devices
- State management using Redux / Context API
- Optimized UI with modern React practices
- Integrated AI chatbot for interactive user assistance
- Weather-based AI-driven outfit recommendation system

## 🛠 Tech Stack

- React
- JavaScript (ES6+)
- Node.js
- Redux / Context API
- Axios
- Vite
- CSS / Responsive Design

## 🔗 API

This project integrates external APIs to fetch real-time data such as weather information.
